## Tech Stack
- **Storefront:** Astro (static) + Stripe Payment Links
- **CMS/Admin:** Directus
- **Blog:** Ghost (SQLite)
- **Email:** Listmonk
- **Social:** Mixpost Lite
- **Database:** PostgreSQL (shared)
- **Proxy:** Traefik (auto SSL)

## Domains (Planned)
