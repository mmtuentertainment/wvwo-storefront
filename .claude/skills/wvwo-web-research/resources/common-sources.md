## Search Query Patterns

### Documentation
```
"[framework] [feature] documentation"
"[framework] [feature] guide [year]"
"how to [task] in [framework]"
```

### Debugging
```
"[exact error message]"
"[framework] [error type] fix"
"[framework] [feature] not working [year]"
```

### Design
```
"[industry] website design inspiration"
"[pattern] UI examples"
"[component] design best practices"
```

### Comparison
```
"[thing A] vs [thing B] [year]"
"best [category] for [use case]"
"[framework] alternatives"
```

## Tips

### Getting Current Info
- Always add year to searches: "Astro 5.0 features 2025"
- Check publish/update dates on articles
- Prefer official docs over third-party when possible

### When Pages Won't Load
- Try WebSearch instead of direct WebFetch
- Look for cached versions or mirrors
- Check if site requires login/paywall

### Maximizing Results
- Be specific in prompts: "Extract the installation steps" not "summarize"
- Request specific formats: "as bullet points" or "as a table"
- Focus on sections: "Focus only on the configuration options"
